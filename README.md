## Setup

1. Run `bundle install` to install dependencies
2. Run `bin/rake db:reset` to create, load and seed db
3. Run `rails s` or `bin/rails s -b 0.0.0.0` to start the server

## Dependencies
* Ruby 2.3.5
* Rails 5.2.4
* Sementic UI
